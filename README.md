# AWS Architect Pro Simulator

Welcome to the AWS 3-Tier Architecture Simulator! 

### ⚠️ Important Notes for Students:
1. **Local Saving:** Your profile and exam history are saved locally in your browser. Do not clear your browser cache if you want to keep your history!
2. **Do Not Refresh:** If you refresh the page *during* an active exam, your current progress will be lost.
3. **Customization:** The hints will show default AWS regions and IP ranges, but the simulator is smart! You can type your own custom Regions, Availability Zones, and CIDR blocks, and the simulator will accept them.
